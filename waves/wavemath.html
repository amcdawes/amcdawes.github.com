Exploring the sin function description of waves

Exercise 1:

  1. Check that f(x,t)= 2.5*sin(0.75*x-0.5*t) in the first row.
  2. Check that g(x,t)=0 in the second row.
  3. Hide all but the f(x,t) function by clicking the colored icon to the left of the other functions.
  4. In the equation for f(x,t) above, what are the numerical values of k, λ, ω, f, and Amplitude?
  5. Check your answer for λ by measuring the wavelength as follows: find the x- position of one of the wavecrests by placing the cursor on the peak and holding down the mouse. In the same way, find the x-position of another wavecrest. Use these two numbers to find the wavelength. Compare this measurement to the value of λ from the equation.
  6. Click on the play button next to the t=0 equation and watch the motion of f(x,t).
  7. Change the sign in the argument, i.e. change sin(k*x-w*t) to sin(k*x+w*t), by changing the sign in the equation.
  8. If the simulation isn't running, click the play button again.
  9. Describe what happens.

Exercise 2:

  1. Stop the animation and set t=0 with the slider
  2. Set f(x,t) = 2.5*sin(0.75*x-0.5*t+1)
  3. Set g(x,t) = 2.5*sin(0.75*x-0.5*t)
  4. How do the graphs of f(x,t) and g(x,t) compare?
  5. Describe the behavior as you change the 1 to 2, 3, 4.
  6. Can you find a value other than 0 that makes f(x,t) line up with g(x,t)? Hint: Think of the easier function sin(x). How much do you need to change x by for sin(x) to repeat?

Exercise 3:

  1. Set g(x,t)=2.5*sin(0.75*x-0.5*t)
  2. Set f(x,t)=2.5*sin(0.75*(x+shift)-0.5*t), where shift is some number. Start with shift=1.
  3. Describe the behavior as you change shift to 2, 3, 4.
  4. What value(s) other than 0 make f(x,t) line up with g(x,t)? What do you notice special about them?
